# SpringBoot + Infinispan 

This SpringBoot base project implements the Infinispan library for Java in the embedded mode. 

It also exemplifies how to manage a data Cache: creating cach√©s, managing their entire lifecycle, grouping its data, updating it, and making differents kinds of querying.

## Needed: 

- Maven 
- Java 8

# Get the project ready to run

## From the project's root:

```
mvn clean
mvn install
mvn package
```

## Run the project:

```
java -jar target/gs-spring-boot-0.1.0.jar
```
